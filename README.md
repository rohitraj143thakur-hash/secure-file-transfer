
# 🔐 Secure File Transfer System

A web-based Secure File Transfer System with AES-256 encryption.

## Features
- AES-256, RSA-2048 Encryption
- Send & Receive Files with Transfer Code
- Password Protection
- Real-time Activity Log
- File Transfer History

## Tech Stack
- HTML5, CSS3, JavaScript

## How to Use
1. Open index.html in browser
2. Select file and encrypt
3. Share 8-digit transfer code
4. Receiver enters code to download
